Bootcamp 2021: Ninjaformer GUI

My first programming task at City, concluding the 2 week 2021 Programming Bootcamp, although I have only spent 3 days programming this. This Processing project show that I can use arrays, loops, mouse and keyboard interaction and geometric transforms. The project contains a GUI library that I made to create the menu system for my ninjaformer game, unfortunately, while the code for laoding tile, sprite and level information exists (JSON, sprite sheet support), I ran out of time before the submission to actually even start on the game. But you can play around with the main code and build your own GUIs too so...
